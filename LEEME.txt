ANTES DE USAR EL PROYECTO DEBES HACER UNAS MODIFICACIONES PARA QUE FUNCIONE EN TU COMPUTADORA


1. composer update

2. composer dump-autoload

3. RENOMBRA EL ARCHIVO .env.backup DE MANERA QUE QUEDE EN ".env"

4. EJECUTA EL COMANDO php artisan key:generate

5. EJECUTA EL COMANDO php artisan migrate

6. YA PODRAS USAR EL PROYECTO (php artisan serve)
